# Design Tokens — Spacing

Kaynak: yildizholding.com.tr CSS (`.page-content`, `.btn`, `#header`, container).

## Scale (px)

| Token | Değer | Kullanım |
| --- | --- | --- |
| `space.1` | 4 | İkon gap |
| `space.2` | 8 | Form field gap |
| `space.3` | 12 | Chip padding |
| `space.4` | 16 | Card iç padding (xs) |
| `space.5` | 20 | Container yatay (`1.25rem`) |
| `space.6` | 24 | Section iç boşluk |
| `space.8` | 32 | Card padding (md) |
| `space.10` | 40 | Bölüm dikey |
| `space.12` | 48 | Sayfa üst/alt (`page-content`) |

## Layout

| Token | Değer | MUI |
| --- | --- | --- |
| `layout.container.maxWidth` | `md` / `lg` | Public: `md`, Internal: `lg` |
| `layout.page.paddingY` | 48px | `py: 6` |
| `layout.section.paddingY` | 40px | `py: 5` |
| `layout.header.height` | 64px | `Toolbar` minHeight (kurumsal site 101px; operasyon UI daha kompakt) |
